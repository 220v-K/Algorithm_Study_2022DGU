# [프로그래머스] 64061. 크레인 인형뽑기 게임



## 풀이

Stack을 이용하면 간단히 풀 수 있는 문제.

```python
from collections import deque
def solution(board, moves):
    stack = deque([])
    answer = 0

    for move in moves:
        for i in range(len(board)):
            if board[i][move-1] != 0:
                stack.append(board[i][move-1])
                board[i][move-1] = 0
                break

        if len(stack) >= 2 and stack[-1] == stack[-2]:
            stack.pop()
            stack.pop()
            answer += 2
            
    return answer
```
